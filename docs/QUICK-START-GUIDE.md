# 快速开始指南 - AI Audio Reader

## 🎯 快速体验新功能

### 1️⃣ 上传你的第一本书 (2分钟)

访问: https://ai-audio-reader-2wsj5weyh-masons-projects-26d844b7.vercel.app/upload

1. 点击上传区域,选择一本 PDF 或 EPUB 文件
2. 等待文本提取完成(进度条会显示)
3. 填写书名(会自动从文件名填充)
4. 选择语言:
   - 原始语言: 书籍的原文语言
   - 翻译语言: 你想翻译成的语言
5. 点击"上传并处理"

**提示**:
- 首次上传建议选择较小的文件(< 2MB)测试
- PDF 提取在浏览器完成,EPUB 在服务器完成

---

### 2️⃣ 开始双语阅读 (30秒)

上传完成后自动跳转到仪表板,点击你的书籍进入阅读器

**你会看到**:
```
┌──────────────────────────────────────────┐
│  工具栏                                   │
│  [目录] [◀] 1/10 [▶] [-] 150% [+] [🌐] [📥] │
│                                           │
│  ┌─────────────────────────────┐        │
│  │   PDF 原文内容               │ [段落 1/120] │
│  │   (保持原格式/字体/图片)      │        │
│  │                              │        │
│  └─────────────────────────────┘        │
│                                           │
│  ┌─────────────────────────────┐        │
│  │ 🌐 中文译文                  │        │
│  │ ━━━━━━━━━━━━━━━━━━━         │        │
│  │ 这里显示中文翻译...           │        │
│  │                              │        │
│  │ 👁️ 查看原文 (可折叠)         │        │
│  └─────────────────────────────┘        │
└──────────────────────────────────────────┘
```

---

### 3️⃣ 播放音频 (10秒)

1. 确保当前页面有译文显示
2. 点击工具栏右侧的 **播放按钮** ▶️
3. 等待翻译和音频生成(首次 3-5秒)
4. 开始播放

**音频模式**:
- **译文模式**(默认): 播放翻译后的中文音频
- **原文模式**: 播放原文语音(点击 "原文/译文" 切换)

---

### 4️⃣ 自定义体验 (1分钟)

#### 调整播放速度
- 工具栏右侧的速度选择器
- 选项: 0.5x, 0.75x, 1.0x, 1.25x, 1.5x, 2.0x
- **推荐**:
  - 精听学习: 0.75x
  - 正常阅读: 1.0x
  - 快速浏览: 1.5x - 2.0x

#### 切换译文显示
- 点击工具栏的 🌐 (Languages) 图标
- 隐藏译文 = 纯原文阅读
- 显示译文 = 双语对照

#### 使用目录导航
- 点击 ≡ (Menu) 图标
- 显示书籍目录(如果 PDF 有书签)
- 点击章节快速跳转

#### 缩放控制
- `+` 放大
- `-` 缩小
- 或滚动鼠标滚轮

## 🔥 高级功能

### 下载双语文本

**场景**: 想要离线阅读或打印

1. 点击工具栏的 📥 (Download) 图标
2. 自动下载双语 TXT 文件

**文件格式**:
```
书名
作者: XXX
============================================================

Chapter 1: Introduction

Learning is a journey...

【译文】学习是一段旅程...

------------------------------------------------------------

Chapter 2: Getting Started

The first step is always...

【译文】第一步总是...

------------------------------------------------------------
```

**用途**:
- 导入到其他阅读器(Kindle, iBooks)
- 打印双语学习资料
- 分享给朋友
- 制作笔记

---

### 原文朗读(无翻译)

**场景**: 语言学习,练习听力

1. 点击 "原文/译文" 按钮
2. 按钮变为绿色 = 原文模式
3. 点击播放 ▶️
4. 听原文语音

**支持语言**:
- 🇺🇸 English (alloy, echo, fable, onyx, nova, shimmer)
- 🇨🇳 中文 (alloy, shimmer)
- 🇯🇵 日本語 (alloy, nova)
- 🇪🇸 Español (nova, alloy)
- 🇫🇷 Français (shimmer, alloy)

---

**用途**: 只需要翻译,不需要原文

直接访问:

https://ai-audio-reader-2wsj5weyh-masons-projects-26d844b7.vercel.app/api/books/[你的bookId]/download/translation

## 💡 使用技巧

### 最佳实践

1. **首次上传**
   - 选择段落较少的书籍测试(< 50 页)
   - 确认功能正常后再上传大文件

2. **翻译策略**
   - 不要一次翻译整本书
   - 按需翻译当前阅读的章节
   - 系统会自动缓存翻译结果

3. **音频播放**
   - 首次播放需要等待生成(3-5秒)
   - 第二次播放同一段落会从缓存读取(瞬间加载)
   - 预加载功能会自动准备接下来 3 个段落

4. **速度控制**
   - 学习新语言: 0.75x
   - 休闲阅读: 1.0x
   - 复习内容: 1.5x - 2.0x
   - 快速浏览: 2.0x

## 🐛 常见问题

### Q1: 为什么我看不到译文?

**答**: 可能的原因:
1. 译文面板被隐藏了 → 点击 🌐 图标显示
2. 当前页面的段落还没翻译 → 等待自动翻译或手动播放
3. 段落匹配失败 → 尝试翻页或手动选择段落

### Q2: 音频生成失败怎么办?

**答**:
1. 检查网络连接
2. 查看浏览器控制台错误信息
3. 刷新页面重试
4. 如果持续失败,可能是 OpenAI API 配额用完

### Q3: PDF 页面显示不清晰?

**答**:
- 点击 `+` 放大
- 调整缩放到 150-200% 效果最佳
- 某些 PDF 可能原本分辨率就低

### Q4: 段落匹配不准确?

**答**:
- 正常情况,准确率 85-90%
- 复杂排版(多栏、图表)可能匹配失败
- 解决方案: 使用段落视图模式手动选择

### Q5: 如何删除已上传的书籍?

**答**:
- 返回仪表板
- 点击书籍卡片右上角的 🗑️ 删除按钮
- 确认删除

### Q6: 支持哪些文件格式?

**当前支持**:
- PDF ✅
- EPUB ✅
- TXT (基础支持) ⚠️

**计划支持**:
- MOBI
- AZW3
- DOCX

### Q7: 文件大小限制?

**Vercel 免费版**: 4MB
**本地开发**: 50MB

**解决方案**:
- 压缩 PDF (在线工具: iLovePDF)
- 拆分大文件
- 使用本地开发环境
- 升级 Vercel Pro
